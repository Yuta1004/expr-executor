# expr-executor

Rust

## コマンド

```
// 実行
cargo run
```

## 実行例

```
// 1
in [0] > 1 + 2
out[0] > 3
in [1] > 1204 + 4 + 10
out[1] > 1218
in [2] > (10+4) * (12/4) + 11
out[2] > 53
in [3] > exit

// 2
in [0] > 10+20
out[0] > 30
in [1] > 30+40
out[1] > 70
in [2] > 50+60
out[2] > 110
in [3] > (*1+*2) / *0
out[3] > 6
in [4] > exit
```

## 実行できる式

- 四則演算(+, -, *, /)を含む式
- ()を含む式
